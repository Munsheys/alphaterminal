# AlphaTerminal 🛡️

> **The Sovereign Trader’s Command Center**

AlphaTerminal is a premium, local-first workstation designed for professional at-home traders. It bridges the gap between performance journaling and tax optimization by providing real-time risk-of-ruin simulations and cross-account wash-sale detection.

## ✨ Features

- 🛡️ **Risk-of-Ruin Engine**: Monte Carlo simulations to visualize account survivability.
- 💸 **Wash-Sale Sovereign**: Detects 30-day window violations across multiple brokers (Robinhood, IBKR, etc.).
- 📉 **Tax-Loss Harvesting**: Real-time suggestions to close losing positions and offset gains.
- 🔒 **Local-First Privacy**: No trade data leaves your browser. Zero cloud storage.
- 📟 **Neo-Terminal UI**: High-density HUD aesthetic for focused execution review.

## 🛠️ Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Logic**: Simple-Statistics (Monte Carlo), PapaParse (CSV)
- **Icons**: Lucide React
- **Animation**: Framer Motion

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/user/alphaterminal.git
   cd alphaterminal
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
